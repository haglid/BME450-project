# Title
Brain MRI Tumor Classification Using Convolutional Neural Networks
## Team members
Hailey Haglid (Haglid)
## Project decription
In this project, we will train a convolutional neural network to classify brain MRI images into tumor categories. We plan to use an online repository from Kaggle.com of labeled brain MRI scans containing classes such as glioma, meningioma, pituitary tumor, and no tumor. The goal of the project is to evaluate how well CNN-based models can distinguish among these classes from medical images.

We will compare two architectures, including a basic custom CNN and a transfer learning model such as ResNet. We will also test several parameters including learning rate, batch size, and image resolution (if applicable). Model performance will be evaluated using training and test accuracy, loss curves, and confusion matrices.
